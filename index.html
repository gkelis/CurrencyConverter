CurrencyConverter by gkelis

Project 2 of WWW Technologies

Steps for successful deployment: 1) Install Apache Tomcat 2) Extract the p2 folder into the “installation_directory\Apache Software Foundation\Tomcat 7.0\webapps” directory 3) Start (or restart) Tomcat service 4) Enter " http://localhost:8080/p2 " in the address bar of any web browser 5) Start playing!! The source code of all the .java files is located in the src folder.
